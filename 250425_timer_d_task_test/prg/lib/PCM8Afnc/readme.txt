==========================================================================
                            本ライブラリの説明
==========================================================================

PCM8A は、デフォルト状態では帰線期間割り込みを許可していません。よって XSP
などの帰線期間割り込みを用いたソフトに対して、割り込みの取りこぼしを発生さ
せてしまいます。

本ライブラリを用いることにより、PCM8A の割り込みマスク設定を、帰線期間割り
込みを許可（または禁止）するように修正することができます。

本ライブラリは XSP 専用に設計されたものではありません。PCM8A の割り込みと
帰線期間割り込みの衝突を回避したい一般的なケースで利用できます。




==========================================================================
                               各関数の説明
==========================================================================

各関数の使用法を以下に説明します。


--------------------------------------------------------------------------

● PCM8A の帰線期間割り込みを許可

書式：void pcm8a_vsyncint_on();

戻り値：無し

機能：帰線期間割り込みを許可するよう、PCM8A の割り込みマスクを変更します。
      PCM8A の常駐判定は関数側で行なっていますので、ユーザー側で行う必要は
      ありません。



--------------------------------------------------------------------------

● PCM8A の帰線期間割り込みを禁止

書式：void pcm8a_vsyncint_off();

戻り値：無し

機能：帰線期間割り込みを禁止するよう、PCM8A の割り込みマスクを変更します。
      PCM8A の常駐判定は関数側で行なっていますので、ユーザー側で行う必要は
      ありません。
